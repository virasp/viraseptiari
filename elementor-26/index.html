About Me

Saya lahir di Denpasar pada tanggal 25 September 2006, merupakan anak pertama dari dua bersaudara dan sedang bersekolah di SMK Negeri 1 Denpasar. Saya memilih jurusan Teknik Informatika dengan jurusan Rekayasa Perangkat Lunak sebagai acuan saya untuk bekerja di masa depan nanti. Selama di sekolah, saya telah belajar berbagai keterampilan seperti membuat dan mendesain sebuah website. 

Saya juga telah memiliki pengalaman magang di PT. Hooki Global Kreasi, dimana saya belajar tentang membuat database dan juga membuat website absensi dengan tampilan yang menarik. Saya berharap dapat terus belajar supaya siap memulai karir saya setelah lulus.
